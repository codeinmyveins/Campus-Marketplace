# Campus Marketplace

**Live Application:** [https://campus-marketplace-zdie.onrender.com/](https://campus-marketplace-zdie.onrender.com/)

Campus Marketplace is a full-stack web application designed to facilitate secure buying and selling within a campus community. The platform enables users to create listings, upload and manage multiple item images, search and filter products, and communicate with sellers in a structured and reliable environment.

The system is built with a strong focus on data integrity, transactional consistency, and secure backend architecture.

---

### Tech Stack

**Frontend:**
HTML, Tailwind CSS, Vanilla JavaScript

**Backend:**
Node.js, Express.js

**Database:**
PostgreSQL

**Storage:**
Supabase Storage (for image management)

---

### Key Technical Features

* OTP-based authentication with secure session handling
* Cookie-based authentication with hashed refresh tokens
* Transaction-safe multi-image upload and deletion
* Concurrency-safe image reordering using row-level locking
* Strict server-side validation and file-type enforcement
* Advanced filtering and sorting powered by PostgreSQL
